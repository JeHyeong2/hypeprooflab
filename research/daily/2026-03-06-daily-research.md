# 펜타곤이 블랙리스트에 올린 AI가 파이어폭스를 지키고 있다

미 국방부가 "공급망 리스크"로 지목한 AI 회사가, 같은 주에 세계에서 가장 많이 쓰이는 브라우저의 보안 취약점 22개를 찾아냈다. Anthropic의 한 주는 그야말로 아이러니의 극치였다. 🟢 Observed

국방부 장관 피트 헥세스는 Anthropic을 공급망 리스크로 공식 지정했고, 방산업체들은 "만약을 위해" Claude 사용을 선제적으로 중단하기 시작했다. CEO 다리오 아모데이는 1,600단어짜리 내부 메모에서 이렇게 썼다 — "우리는 트럼프에게 기부하지 않았고, 독재자식 찬사를 보내지도 않았다." 이 발언의 맥락은 명확하다. OpenAI가 백악관에 줄을 서는 동안, Anthropic은 정치적 중립을 택했고, 그 대가를 치르고 있다는 것이다. 🟢 Observed

그런데 흥미로운 반전이 있다. 블랙리스트 이후 Claude의 일일 신규 가입자 수는 오히려 역대 최고치를 경신하고 있다. AppFigures 데이터에 따르면 미국, 캐나다, 유럽 전역에서 앱스토어 무료 및 AI 카테고리 1위를 기록 중이다. 정부가 때리니 시장이 안아주는 형국이다. 🟢 Observed

같은 주, Anthropic은 Mozilla와의 협업 결과를 공개했다. Claude Opus 4.6가 Firefox에서 22개의 보안 취약점을 발견했고, 이 중 14개가 고위험으로 분류됐다 — 이는 2025년 전체 고위험 Firefox 취약점의 거의 5분의 1에 해당한다. 특히 JavaScript 엔진 탐색 시작 20분 만에 Use After Free 취약점을 찾아낸 것은 인간 보안 연구자가 며칠에서 몇 주 걸리는 작업을 AI가 압축한 사례다. 발견된 취약점들은 Firefox 148.0에서 수억 명의 사용자에게 패치로 배포됐다. 🟢 Observed

한편, OpenAI도 보안 영역에 본격 진출했다. Codex Security라는 새로운 리서치 프리뷰 에이전트를 출시해 앱 보안 취약점 탐지와 수정에 특화된 도구를 선보였다. 오픈소스 개발자들에게는 Codex Open Source Fund를 통해 6개월간 ChatGPT Pro 구독과 함께 Codex Security 접근권을 제공한다. 더 주목할 것은 OpenAI가 GitHub 경쟁 서비스까지 개발 중이라는 보도다. 최근 GitHub 장애에 촉발된 이 프로젝트는 아직 수개월 남았지만, 완성되면 OpenAI가 최대 투자자 Microsoft와 직접 경쟁하는 기묘한 구도가 된다. 🔵 Supported

EU에서는 Meta가 WhatsApp에서 경쟁사 AI 챗봇을 허용하겠다고 발표했다. 유럽연합 반독점 규제를 달래기 위한 조치로, 12개월간 "유료로" WhatsApp Business API를 통해 범용 AI 챗봇을 지원한다. 이전까지 ChatGPT, Copilot 등이 WhatsApp에서 차단당했던 것과 비교하면 의미 있는 변화이나, "유료"라는 단서가 붙은 만큼 실질적 개방이 될지는 두고 봐야 한다. 🟡 Speculative

**So what?** AI 산업의 권력 지형이 빠르게 재편되고 있다. 기술력과 정치력이 분리된 세계에서, Anthropic은 기술로 증명하고 정치로 벌 받는 역설을 살고 있다. OpenAI는 코딩 인프라까지 삼키려 하고, Meta는 규제 앞에서 문을 열었다. 이 모든 일이 단 한 주에 일어났다.
