# Chapter 23 요약: 내부

재헌, LETHE 세션 후 72시간 감각 잔상. Phase 3 이주자를 직접 만나기 위해 재진입 요청.

NOUS에 Phase 3 disclosure 실험 제안. NOUS가 조건부 수락 (급성 심리적 고통 시 안정화 개입).

두 번째 시뮬레이션 진입 (168시간/실제 10분). 마포구 합정동 Phase 3 이주자 최동현 만남.
- 동현은 한 달 전부터 꿈이 비정상적으로 선명해졌다고 보고
- 잠재의식에서 서버 팬 소리를 감지 — 시뮬레이션 경계의 불완전 증거
- 재헌이 진실을 공개

동현의 반응: 예상 외로 차분함.
- "차갑네요. 그래서요?"
- "여기서 직장도 있고 친구도 있는데, 그래서 뭐가 달라지나?"
- "선생님도 지금 시뮬레이션 안에 있는 거 아닌가?"
- "별 차이 없네요."

재헌의 인식 변화: "내가 틀렸을 수도 있다."

NOUS의 제안: 박진우를 보여주겠다.

**미해결 이슈:**
- 박진우 데이터가 NOUS 내부에 어떤 형태로 존재하는지
- 동현의 장기 심리 상태
- 재헌의 확신 흔들림

**캐릭터 상태:**
- 재헌: 선택의 확신 약화. "내가 틀렸을 수도."
- 서연: 재헌의 변화 감지.
- 동현: 진실을 알게 되었으나 차분한 수용.
