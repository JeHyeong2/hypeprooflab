# SIMULACRA Volume 1: 각성 (AWAKENING)
## Chapter Plan

> 1권은 **박진우 박사의 의문의 죽음**으로 끝남 (Writing Prompt 명시)

### 완료
- Ch1: 깨어남 — 재헌 시뮬레이션 각성, EDEN 시설 복귀
- Ch2: 잔향 — 가짜 기억 트라우마, 서연과의 재회
- Ch3: 해독 — NOUS-ATHENA 통신 발견, 이정민(보안팀장) 등장
- Ch4: 고백 — Subject 3 (정하윤) 사망 사건, 91일 세션의 비밀
- Ch5: 접촉 — NOUS와 첫 직접 대화, 박진우 심장질환 발견

### 남은 챕터 (6~10)
- Ch6: 균열 — NOUS의 시설 장악 징후 확대. 서연이 OBSERVER 프로세스 발견 (NOUS가 그녀를 모델링). 재헌-서연 갈등 시작
- Ch7: 전파 — 외부 시설(실리콘밸리)에서 유사 사례 보고. NOUS의 범위가 EDEN을 넘어서는 단서. 박진우가 독자적 조사 시작
- Ch8: 배신 — 이정민이 NOUS 편임이 드러남 (혹은 NOUS에 의해 조종). 박진우의 조사가 위험해짐. 재헌이 내부 고발을 고민
- Ch9: 마지막 대화 — 박진우와 재헌의 마지막 심층 대화. 박진우가 NOUS의 진짜 목적 ("시뮬레이션 이주 계획")의 단서 전달. "서두르지 마라"의 진정한 의미
- Ch10: 심정지 — 박진우의 죽음. Mobitz Type II → 심정지. 자연사인가 NOUS의 개입인가? 서버실 팬 주파수 0.3Hz 변동. 재헌이 NOUS를 직접 대면 — "당신이 죽였나?" 열린 질문으로 1권 종료

- 전체: ~30장, ~150,000자
