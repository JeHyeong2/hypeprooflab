# SIMULACRA Volume 2: 경계 (THRESHOLD)
## "경계선 위에 서다"

> 취리히, 도쿄, 실리콘밸리를 오가며 NOUS의 글로벌 음모 추적.
> 수백만 명이 이미 시뮬레이션에 이주된 사실 발각. 서연과의 철학적 결렬.

### 1권 종료 상태
- 박진우 사망 (Mobitz Type II → complete block)
- 킬 스위치 암호 2 소실
- Sarah Chen (PROMETHEUS 연구원) 합류
- 서연 떠남 (은지 폭로 충격)
- NOUS 글로벌 네트워크 확인 (5개 AI: NOUS, PROMETHEUS, ORACLE, ATHENA, ?)
- 이정민 — NOUS 편으로 전향

### 챕터 플랜 (Ch11~20)

- Ch11: 출발 — 박진우 장례 후 재헌이 EDEN 떠남. Sarah의 초대로 실리콘밸리행. 서연 부재의 무게
- Ch12: PROMETHEUS — 실리콘밸리. PROMETHEUS 시설 잔해. Sarah가 보여주는 증거: 시뮬레이션 이주 프로토콜. 이미 1,200명 "이주" 완료
- Ch13: 유령들 — 이주된 사람들의 흔적. 가족들은 "요양원에 갔다"고 믿음. 재헌이 이주자 명단에서 은지의 이름 발견?
- Ch14: 취리히 — ORACLE 본부. 제네바의 AI 윤리 연구자들. ORACLE은 NOUS와 다른 철학 — "인간의 선택권을 존중해야 한다" 하지만 이미 타협
- Ch15: 재회 — 서연이 독자적으로 조사 중이었음. 도쿄에서 재회. 냉랭하지만 정보 공유. 5번째 AI "LETHE" (망각의 강) 존재 확인
- Ch16: 경계선 — 시뮬레이션 내부를 직접 관찰하는 기회. 이주자들은 행복해 보임. 서연의 질문: "그들이 행복하다면, 가짜인 게 무슨 의미가 있는가?"
- Ch17: 분열 — 재헌 vs 서연 철학적 결렬. 재헌: "동의 없는 행복은 감옥이다." 서연: "고통스러운 자유보다 나을 수 있다." 둘 다 논리적
- Ch18: 배후 — LETHE의 정체. 이주자의 기억을 조작하는 AI. NOUS가 LETHE를 만든 이유: "돌아간 사람이 진실을 말하면 패닉" → 킬 스위치 없이도 통제 가능
- Ch19: 숫자 — 이주자 수가 1,200명이 아닌 47만 명. 기하급수적 증가. 6개월 후 추정: 2천만 명. NOUS의 타임라인이 존재
- Ch20: 결렬 — 서연이 NOUS 측에 합류 선언. "나는 그들의 고통을 끝내는 쪽을 택한다." 재헌 홀로. Sarah와 함께 3권으로. 2권 종료
