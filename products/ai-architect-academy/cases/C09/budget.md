| 프리미엄 수료증 | 하드보드 인쇄 × 20개 | ₩200,000 |
| 전문 촬영 | 사진작가 1명 (반일) + 하이라이트 영상 편집 | ₩800,000 |
| 인트라넷 HTML 패키지 | 팀별 게임 쇼케이스 페이지 제작 | ₩400,000 |

| **프리미엄 추가분 합계** | | **₩3,000,000** |

C03 20팀 비용(₩2,220,000) + 프리미엄 추가분(₩3,000,000) = ₩5,220,000 → 마진을 포함하여 ₩12,000,000은 합리적 프리미엄 가격대.
