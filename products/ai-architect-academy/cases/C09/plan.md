# C09 — IT 대기업 신입사원 자녀 온보딩 특별 워크숍 (B2B Premium)
## Full Business Plan

---

## Executive Summary

C03(기업 CSR 워크숍, ₩5~8M)의 프리미엄 상위 티어로, IT 대기업(삼성전자, SK하이닉스, 현대오토에버, 네이버, 카카오 등)의 신입/경력 사원 Family Day 또는 Welcome Package 프로그램에 "AI 교육 체험"을 삽입하는 B2B 프리미엄 상품이다. 회당 ₩8,000,000~12,000,000(10~20팀 기준)으로, C03 대비 50~100% 높은 단가를 기업 복지 예산에서 집행한다. IT 업계 부모는 AI 리터러시에 대한 이해도와 투자 의지가 일반 기업 대비 월등히 높아 프리미엄 포지셔닝이 자연스럽다. "삼성 임직원 자녀가 AI로 만든 게임"이라는 SNS 바이럴 콘텐츠가 자동 생성되며, 기업 인사 브랜딩 소재로도 활용된다. 연 2~3개 IT 대기업 × 2~4회 세션으로 ₩48,000,000~144,000,000 연매출을 전망한다.

## 1. Premium Positioning — Why ₩8~12M vs C03's ₩5~8M?

### 1.1 C03 vs C09 비교

| 항목 | C03 (Standard B2B) | C09 (Premium B2B) |
|------|--------------------|--------------------|
| 대상 기업 | 전 업종 대기업 | IT/테크 대기업 한정 |
| 예산 출처 | CSR 예산 | 복지(welfare) 예산 |
| 회당 단가 | ₩5,000,000~8,000,000 | ₩8,000,000~12,000,000 |
| 팀 규모 | 20~30팀 | 10~20팀 |
| 팀당 실질 단가 | ₩170,000~250,000 | ₩400,000~1,200,000 |
| 커리큘럼 커스터마이징 | 오프닝/클로징 기업 로고 수준 | IT 업종 맞춤 커리큘럼 + 기업 기술 맥락 반영 |
| 결과물 | 게임 URL + 수료증 | 게임 URL + 프리미엄 수료증 + 브랜디드 USB + 사내 인트라넷 게시 |
| 촬영 | 단체 사진 | 전문 사진 + 영상 촬영 (기업 브랜딩용) |
| 사후 관리 | NPS 설문 | NPS + HR 보고서 패키지 + 사내보 기사 초안 |

### 1.2 프리미엄 단가 정당화 (₩8~12M)

**1) IT 업종 맞춤 커리큘럼 커스터마이징**
- Part 1 오프닝: "여러분 부모님이 만드는 반도체/클라우드/AI 서비스도 이런 과정으로 설계됩니다" — 기업 기술 맥락과 커리큘럼을 연결하는 스토리라인 개발 (기업당 D-30에 사전 인터뷰 진행)
- Part 2 기획서: IT 기업 도메인 주제 예시 제공 (예: "AI가 배달을 최적화하는 게임", "우주에서 반도체 칩을 지키는 게임")
- 기업별 사전 인터뷰 + 커스터마이징에 Filamentree PM 40시간 추가 투입

**2) 프리미엄 결과물 패키지**
- 브랜디드 USB 드라이브: 기업 로고 각인, 완성 게임 + 제작 과정 영상 저장
- 프리미엄 수료증: 하드보드 인쇄 + 기업 CEO/CHRO 직인 (기업측 협의)
- 사내 인트라넷 게임 쇼케이스 페이지: 팀별 게임 URL + 제작 스토리 + 사진을 기업 내부망에 게시 가능한 HTML 패키지 납품

**3) 전문 촬영 + 브랜딩 콘텐츠**
- 전문 사진작가 1명 현장 파견
- 편집된 하이라이트 영상 (3~5분) D+14 납품
- 기업 사내보/뉴스레터용 기사 초안 제공

**4) HR 보고서 패키지**
- 참가자 만족도 분석 리포트 (NPS, 자유 응답 분석)
- 기업 복지 프로그램 성과 보고 양식에 맞춘 정량 데이터
- 재실행 제안서 (다음 세션 개선사항 포함)

### 1.3 IT 기업 부모의 차별화된 니즈

IT 업계 임직원은 다른 업종 대비 뚜렷한 차별점을 가진다:

| 특성 | 일반 기업 임직원 | IT 기업 임직원 |
|------|-----------------|---------------|
| AI 이해도 | 뉴스 수준 | 실무 활용 경험 보유 |
| 자녀 AI 교육 투자 의지 | 중간 ("해봤으면 좋겠다") | 매우 높음 ("필수라고 확신") |
| 워크숍 기대치 | "AI가 뭔지 알려주세요" | "AI를 어떻게 지휘하는지 경험하게 해주세요" |
| 가정 내 확장 가능성 | 낮음 (부모가 모름) | 높음 (부모가 직접 복습 가능) |
| 워크숍 후 피드백 | 만족도 중심 | 커리큘럼 구조에 대한 구체적 피드백 (향후 개선 소스) |

IT 부모의 높은 기대치는 리스크(커리큘럼 수준이 기대 이하일 경우 불만 즉시 표출)이면서 동시에 기회(성공 시 구체적이고 설득력 있는 추천 확보)다.

### 1.4 복지 예산 vs CSR 예산

| 항목 | CSR 예산 (C03) | 복지 예산 (C09) |
|------|---------------|----------------|
| 결재 속도 | 느림 (CSR위원회 승인 필요) | 빠름 (HR팀 자체 집행 가능) |
| 예산 규모 | 기업별 편차 큼, 경기 민감 | 1인당 복지 포인트 기반, 안정적 |
| 결재 라인 | CSR 부서장 → 경영진 | HR팀장/복지팀장 → 인사담당임원 |
| 구매 절차 | RFP/입찰 가능 | 수의계약 또는 복지 프로그램 선정 |
| 집행 유연성 | 사전 계획 필수 (연간) | 분기/월별 유연 집행 가능 |
| KPI | CSR 보고서 지표 (봉사시간, 수혜자 수) | 임직원 만족도, 복지 프로그램 참여율 |

**핵심 인사이트:** 복지 예산은 CSR 예산보다 집행이 빠르고 유연하다. Family Day 같은 단발 이벤트에 ₩8~12M을 집행하는 것은 복지 예산 규모에서 일반적인 범위다. (참고: 대기업 임직원 1인당 연간 복지 포인트 ₩1,200,000~3,600,000 수준, 자녀 교육은 복지 포인트 사용 인기 항목)

## 2. Curriculum Adaptation (IT Premium)

### 2.1 원칙: 4-Part 구조 보존 + IT 업종 딥 커스터마이징

기본 커리큘럼 4-Part(240분)를 유지하되, C03보다 한 단계 깊은 업종 맞춤을 적용한다.

### 2.2 IT 업종 커스터마이징 포인트

| Part | 기본 커리큘럼 | C03 적응 | C09 프리미엄 적응 |
|------|------------|---------|-------------------|
| Part 1 (관점 전환) | AI 라이브 데모 | 기업 로고 오프닝 | IT 기업 기술 사례 연결 ("부모님이 만드는 AI 서비스의 설계 과정도 이와 같아요"), 부모 참여형 퀴즈 추가 |
| Part 2 (협업) | 브레인스토밍 + 기획서 | 기업 가치 연결 주제 | IT 도메인 게임 주제 카드 (클라우드, 반도체, 모빌리티 등), 부모가 "기술 자문 역할" 추가 부여 |
| Part 3 (제작) | AI 게임 제작 | 변경 없음 | IT 부모의 높은 기술 이해도를 활용 — 부모 역할 확대 ("Context Engineering 코칭"), Advanced 도전 과제 추가 |
| Part 4 (발표) | 발표 + 시상 | 임원 인사말 5분 | CHRO/HR VP 축사 + "Best IT Family" 특별상 추가 + 기업 브랜딩 촬영 세션 10분 |

### 2.3 IT 부모 역할 강화 — "기술 자문 역할"

C03에서 부모는 "구조화" 역할(Integrator)에 그치지만, C09에서는 IT 부모의 도메인 지식을 적극 활용한다:

- Part 2: 부모가 자신의 업무 경험에서 게임 아이디어를 제안 ("내가 만드는 클라우드 서비스처럼, 서버가 공격받으면 방어하는 게임은 어때?")
- Part 3: 부모가 "Context Engineering 코칭" — AI에게 지시할 때 기술적 맥락을 추가하는 방법을 자녀에게 직접 시연
- 이 구조는 "부모님도 배우는 교육"에서 "부모님이 선생님이 되는 교육"으로 포지션 전환 — IT 부모의 자존감 충족

### 4.2 바이럴 잠재력 — "삼성 임직원 자녀가 만든 게임"

IT 기업 워크숍에서 자동 생성되는 바이럴 콘텐츠:
- 자녀가 만든 게임 URL → 기업 인트라넷 게시 → 임직원 간 공유
- "우리 팀 자녀 게임 1등" 같은 사내 경쟁 → 다음 세션 참가 수요 자체 생성
- 부모 SNS 인증: "삼성에서 아이와 함께 AI 게임을 만들었다" → 외부 노출
- 기업 채용 브랜딩: "우리 회사는 임직원 자녀에게 AI 교육까지 제공합니다"

**NDA 주의:** IT 기업 특성상 SNS 공유 범위는 사전 협의 필수. 기업 로고 노출 제한 가능. 대안으로 "IT 기업 Family Day"로 모자이크 처리 후 Filamentree/동아 마케팅 소재로 활용.
