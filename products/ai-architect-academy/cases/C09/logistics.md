## 1. IT Company Campus Requirements

### 1.1 기업 캠퍼스 보안 — C03 대비 강화 사항

IT 대기업(삼성, SK, 현대 등)의 보안 정책은 일반 기업보다 한 단계 높다:

| 보안 항목 | C03 (일반 기업) | C09 (IT 대기업) |
|----------|---------------|----------------|
| 출입 절차 | 방문자 목록 D-5 제출 | 방문자 목록 + 신분증 사본 + 목적 기재, D-10 제출 |
| 장비 반입 | 사전 알림 수준 | **장비 목록 + 시리얼 번호 사전 등록**, 보안 게이트 검사 |
| 노트북 | 참가자 자체 장비 사용 | 기업 보안 소프트웨어 설치 노트북은 외부 인터넷 제한 가능 → **참가자 개인 노트북 지참 필수** 사전 안내 |
| Wi-Fi | 기업 게스트 Wi-Fi | IT 기업 게스트 Wi-Fi도 접속 절차 복잡 (MAC 등록, 보안 동의) → **LTE 라우터 자체 반입 필수** |
| 촬영 | 사전 협의 | **촬영 허용 구역 사전 서면 합의 필수**; 일부 IT 기업은 교육장 내 개인 기기 카메라 테이핑 요구 가능 |
| USB/외부 저장장치 | 자유 | **USB 반입 사전 승인 필요**; 브랜디드 USB는 기업 보안팀 사전 검사 가능 |
| 네트워크 | 기업 Wi-Fi 사용 | **AI 서비스 접속이 기업 방화벽에 차단될 수 있음** → 사전 화이트리스트 요청 또는 LTE 사용 |

### 1.3 네트워크 구성 (권장)

```
[참가자 노트북 20대] --> [LTE 라우터 1-3] --> [인터넷]
                    \
                     --> [기업 게스트 Wi-Fi (백업)] --> [인터넷]

[강사 노트북] --> [프로젝터/대형 모니터]
             --> [LTE 라우터 (전용)]
```

**LTE 라우터 사양:** 동시 접속 10대 이상, LTE Cat.6 이상 (다운로드 300Mbps)
**필요 대수:** 2대 (참가자용) + 1대 (강사 전용) = 3대
**비용:** 렌탈 ₩30,000/대/일 × 3대 = ₩90,000 (비상 예비비에서 집행)

## 3. Company Intranet Game Showcase

### 3.1 인트라넷 게시 패키지

워크숍 종료 후 D+14에 기업에 납품하는 HTML 패키지:

```
/family-day-showcase/
  index.html           -- 메인 페이지 (팀 리스트 + 하이라이트 사진)
  /teams/
    team-01.html       -- 팀별 페이지 (게임 URL, 기획서, 제작 스토리, 사진)
    team-02.html
    ...
  /assets/
    photos/            -- 편집된 워크숍 사진
    videos/            -- 하이라이트 영상 (3-5분)
    style.css          -- 기업 CI 컬러 적용 스타일
```

### 3.2 인트라넷 게시 워크플로우

| 단계 | 시점 | 담당 | 산출물 |
|------|------|------|--------|
| 1. 게임 URL 수집 | D-Day | 조교 | 팀별 GitHub Pages URL 리스트 |
| 2. 사진/영상 편집 | D+7 | 전문 촬영팀 | 편집본 |
| 3. HTML 패키지 제작 | D+10 | Filamentree 개발팀 | showcase 폴더 |
| 4. 기업 HR 리뷰 | D+12 | 기업 HR팀 | 수정 요청 (사진 삭제 등) |
| 5. 최종 납품 | D+14 | Filamentree | zip 파일 전달 |
| 6. 인트라넷 게시 | D+14~21 | **기업 IT팀** | 사내 인트라넷 페이지 |

**중요:** 인트라넷 실제 게시는 기업 IT팀이 담당. Filamentree는 HTML 패키지만 납품. 기업 인트라넷 기술 스택(SharePoint, Confluence, 자체 시스템)에 관계없이 작동하는 정적 HTML 제공.

### 3.3 게임 호스팅

| 옵션 | 장점 | 단점 |
|------|------|------|
| GitHub Pages (기본) | 무료, 간단, 외부 접근 가능 | 기업 보안 정책에 따라 접근 차단 가능 |
| 기업 내부 서버 | 보안 정책 준수 | 기업 IT팀 협조 필요, 셋업 시간 |
| Filamentree 서버 | 통제 가능 | 호스팅 비용 발생, 외부 의존 |

**권장:** GitHub Pages 기본 → 기업이 내부 호스팅 요청 시 HTML 파일 일괄 납품.
