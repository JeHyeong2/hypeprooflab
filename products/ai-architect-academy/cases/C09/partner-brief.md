## 2. 왜 이 프로그램인가?

### IT 업계 부모의 고민

> *"AI가 세상을 바꾸고 있다는 걸 누구보다 잘 알지만, 우리 아이에게 어떻게 알려줘야 할지 모르겠습니다."*

귀사 임직원은 AI를 매일 업무에서 활용하는 전문가입니다. 하지만 자녀에게 AI 역량을 전달하는 방법은 다릅니다. 코딩 학원에 보내는 것은 이미 구세대 접근이고, AI 시대에 필요한 것은 **"AI를 지휘하는 설계자(Architect)"** 로서의 사고력입니다.

### 이 프로그램이 다른 점

| 기존 프로그램 | Future AI Leader's Academy |
|-------------|---------------------------|
| 아이만 수강, 부모는 대기실 | **부모-자녀가 한 팀** (Co-Founding 모델) |
| 코딩 문법 암기 | **AI에게 지시하는 법** (Context Engineering) |
| 이론 강의 6주 | **4시간 만에 실제 게임 완성** |
| 완성물 없음 | **플레이 가능한 게임 URL** → 사내 인트라넷 공유 가능 |
| 표준 커리큘럼 | **귀사 기술 맥락에 맞춤** (반도체, 클라우드, 모빌리티 등) |

---

## 3. 프로그램 흐름 (4시간 15분)

```
Part 1: 관점의 전환 (45분)
  "코딩하는 사람이 아니라, AI를 지휘하는 설계자가 됩니다"
  - AI 라이브 데모 (30초 만에 게임 생성)
  - 귀사 기술 사례와 연결 ("부모님이 만드는 서비스도 이런 설계 과정을 거칩니다")
      |
      v
Part 2: 협업과 조율 (50분)
  "아이의 상상력 + 부모의 현실 감각 = 기획서 완성"
  - 아이: "하늘 나는 고양이 게임 만들래!"
  - 부모: "날개는 어떤 버튼으로 조종해?" (IT 부모의 기술 자문 역할)
      |
      v
Part 3: 지휘와 소통 (90분)
  "AI에게 지시하여 실제 게임 제작"
  - 10초 피드백 루프: 지시 → 결과 확인 → 수정 → 반복
  - 부모가 Context Engineering 코칭 (IT 전문성 활용)
      |
      v
Part 4: 성취와 비전 (70분)
  "팀 발표 + 시상 + CHRO 축사 + 브랜딩 촬영"
  - 5개 시상 카테고리 (모든 팀이 하나 이상 수상)
  - 'Future AI Leader' 프리미엄 수료증 수여
  - 전문 촬영 (기업 브랜딩 소재)
```

## 4. 귀사가 얻는 것 (ROI)

### 4.1 임직원 만족도 향상

| 지표 | 기대치 | 근거 |
|------|--------|------|
| 참가자 NPS | 92%+ | 기존 워크숍 실적 기반 |
| "다시 참가하고 싶다" | 85%+ | 기존 워크숍 학부모 피드백 |
| "회사 복지 만족" 상승 | 측정 가능 | 사전/사후 복지 만족도 비교 |

### 4.2 기업 브랜딩 소재

| 소재 | 활용처 |
|------|--------|
| "임직원 자녀가 AI로 만든 게임" 쇼케이스 | 사내 인트라넷, 사내보 |
| 하이라이트 영상 (3~5분) | 채용 브랜딩, SNS, 기업 소개 |

| 사내보 기사 초안 (800자) | 기업 뉴스레터 즉시 게재 가능 |

| 항목 | 수치 |
|------|------|
| 1인당 복지 포인트 연간 | ₩1,200,000~3,600,000 (대기업 평균) |
| 자녀 교육 복지 항목 사용률 | 상위 3 인기 카테고리 |
| 본 프로그램 1인당 비용 | ₩200,000~400,000 (팀당 ₩400K~800K / 2인) |
| 일반 Family Day 1인당 비용 | ₩300,000~500,000 (업계 벤치마크) |
| 비교 | **동종 행사 대비 동일하거나 저렴한 1인당 비용에 교육 성과물까지 제공** |
| 복지 포인트 대비 | 연간 포인트의 11~33% 수준 |

### 4.4 경쟁사 대비 차별화

현재 IT 대기업의 임직원 자녀 프로그램:
- **삼성**: 임직원 자녀 캠프 (일반 체험형)
- **SK**: SK 행복나눔재단 연계 봉사 프로그램
- **LG**: LG 사이언스홀 방문 체험

→ **AI 교육 + 부모-자녀 협업 제작 + 실제 결과물** 조합은 현재 어느 기업도 운영하고 있지 않습니다.

## 8. C03 Standard → C09 Premium 업그레이드

이미 Standard CSR 워크숍(C03)을 경험하신 경우, Premium으로 업그레이드하실 수 있습니다:

| 항목 | C03 Standard | C09 Premium |
|------|-------------|-------------|
| 커리큘럼 | 범용 기업 맞춤 | IT 업종 딥 커스터마이징 |
| 결과물 | 게임 URL + 기본 수료증 | 게임 URL + 브랜디드 USB + 프리미엄 수료증 + 인트라넷 쇼케이스 |
| 촬영 | 단체 사진 | 전문 촬영 + 하이라이트 영상 |
| HR 보고서 | NPS 결과 공유 | 정량 분석 리포트 + 사내보 기사 초안 |
| 부모 역할 | 구조화 지원 | 기술 자문 역할 (IT 전문성 활용) |
