# C06 — 어린이동아 구독자 멤버십 워크숍 (구독자 락인)
## Full Business Plan

---

## 1. Executive Summary

어린이동아의 유료 구독부수가 8만 부에서 5.9만 부로 감소하는 추세에서, Future AI Leader's Academy 워크숍을 '구독 특전'으로 포지셔닝하여 이탈 방어 락인 메커니즘을 구축한다. 구독자 가정은 ₩100,000/팀 할인가로 참여하며, 비구독자는 ₩200,000/팀 정가를 지불한다. 분기 1회 구독자 우선 오픈 → 갱신율 KPI 측정 → 구독 패키지 번들화 결정이라는 단계적 검증 구조를 따른다. 워크숍 중 아이들이 제작한 게임은 어린이동아 지면·앱에 독자 생성 콘텐츠(UGC)로 게재되어 편집부 추가 부담 없이 콘텐츠를 확보하는 이중 효과를 낸다. Filamentree(HypeProof AI Lab)는 커리큘럼·강사를 제공하고, 동아이지에듀는 구독 인프라·채널·공간을 제공하는 역할 분리 구조다.

## 2. Case Context

| 항목 | 내용 |
|------|------|
| **케이스 ID** | C06 |
| **파트너** | 동아이지에듀 (어린이동아 운영팀) |
| **대상 모수** | 어린이동아 유료 구독 가정 59,000부수 |
| **참가자** | 초등 4~6학년 + 부모 (팀 단위) |
| **장소** | 동아미디어센터 (서울) + 분기별 지방 거점 순회 |
| **가격 구조** | 구독자 ₩100,000/팀 / 비구독자 ₩200,000/팀 |
| **운영 주기** | 분기 1회 (연 4회) |
| **전략 목표** | 구독 갱신율 향상 + 신규 구독 유입 + UGC 콘텐츠 확보 |

### 핵심 제약 조건

1. **구독자 할인 (₩100,000)**: 구독 특전으로서의 체감 가치가 핵심. 마케팅 예산으로 ₩100,000 차액을 처리하므로 Filamentree 수익 보전 방식 계약 전 협의 필요.
2. **이탈 방어 기제**: 워크숍 경험이 구독 갱신 동기로 작동해야 함. 워크숍 후 60일 내 갱신율 데이터 수집 및 대조군(미참가 구독자) 비교 체계가 필수.
3. **무광고 홍보 채널**: 어린이동아 지면 + 앱 내 홍보를 추가 광고비 없이 활용. 모집 CAC(고객 획득 비용)를 제로 수준으로 유지.
4. **UGC 콘텐츠 계약**: 아이들 게임의 지면/온라인 게재는 보호자 동의서(개인정보·저작권 포함) 선제적 설계 필요. 등록 시 온라인 동의 수령 + 현장 재서명 이중 구조.
5. **지방 순회**: 서울 외 지역 구독자 커버리지 확대. 회차별 별도 장소 섭외 및 물류 계획 수립. 지방 행사는 Q3부터 도입.

## 3. Curriculum Adaptation

### 원칙
기본 4-part 구조(관점의 전환 → 협업과 조율 → 지휘와 소통 → 성취와 비전)를 유지하되, **어린이동아 구독 독자 정체성**과 **게임 어린이동아 게재 특전**을 전면에 배치한다.

### 시간 조정

| Part | 기본 커리큘럼 | C06 조정 | 변경 이유 |
|------|------------|---------|----------|
| Part 1: 관점의 전환 | 40분 | **35분** | 초등 4~6학년 집중력 고려, 아이스브레이킹 압축. 어린이동아 게재 특전 공지로 동기 선제 확보. |
| Part 2: 협업과 조율 | 50분 | **45분** | 어린이동아 주제 기반 기획서 템플릿 활용으로 브레인스토밍 시간 단축. |
| Part 3: 지휘와 소통 | 90분 | **95분** | 실습 시간 확대. 완성도 높은 게임이 게재 가치를 갖기 위해 +5분. |
| Part 4: 성취와 비전 | 60분 | **65분** | "어린이동아 게재 선정" 세리머니 추가 (10분). |
| **합계** | **240분** | **240분** | — |

### 콘텐츠 수정 사항

#### Part 1 수정 (35분) — 관점의 전환
- **오프닝 특전 공지**: "어린이동아 구독자 특전 카드" 배부. "오늘 만든 게임이 어린이동아에 실릴 수 있습니다"를 첫 5분에 공지. 동기 부여 극대화.
- **AI 데모 주제 현지화**: 어린이동아 실제 기사 소재(우주탐사, 멸종위기 동물, 환경 오염)로 게임 생성 시연. 독자에게 친숙한 소재로 진입 장벽 제거.
- **토론 예시 교체**: "기자 = 벽돌 나르는 사람 / 편집장 = 설계도 그리는 사람" 메타포 활용. 어린이동아 독자에게 익숙한 뉴스룸 언어 사용.
- 팀 구성·역할 배정은 기본 커리큘럼 동일.

#### Part 2 수정 (45분) — 협업과 조율
- **기획서 워크시트 교체**: 어린이동아 독자 전용 "뉴스 주제 게임화" 템플릿. 예: "기후변화를 막는 게임", "우주 탐험 게임", "역사 속 영웅 게임".
- **샘플 게임 선정**: 기존 실적(SPEC.md §9) 중 어린이동아 주제에 근접한 2개 선정 시연.
- **부모 가이드 카드 강화**: 초등 4~6학년 특화 구체화 질문 20선 추가. 예: "이 게임에서 악당은 어떻게 움직여?", "레벨이 올라가면 뭐가 달라져?"
- 휴식 10분 유지.

#### Part 3 수정 (95분) — 지휘와 소통
- **게재 기준 브리핑** (5분 추가, 14:30~14:35): "어린이동아에 실리는 게임 기준 — 창의적 주제 + 플레이 가능 + 제목에 스토리". 아이들의 완성 동기 극대화.
- **10초 피드백 루프** 유지. 조교가 "발표 가능한 게임" 완성에 집중 유도.
- **"포기의 미덕" 훈련** 유지. 단, 15:50 "완성 모드" 전환 신호를 강화.
- 오프라인 백업 템플릿(USB)은 어린이동아 소재 기반으로 사전 제작.

#### Part 4 수정 (65분) — 성취와 비전
- **팀별 발표** (30분): 기존 3~5분 형식 유지. 발표 항목에 "이 게임의 주인공은 어린이동아 기사 어디서 왔나요?" 질문 1개 추가.
- **"어린이동아 편집장 선정" 세리머니** (10분 추가): 어린이동아 편집 담당자가 현장에서 기준(창의성·완성도·스토리) 발표 후 1~2팀 선정. 나머지 팀은 온라인 갤러리 게재 옵션.
- **시상 카테고리**: 기존 5개 유지 + "어린이동아 스타 기자상" (최고 스토리텔링팀) 추가.
- **수료증 문구**: "Future AI Leader's Academy 수료 — 어린이동아 독자 특전 과정" 명기.
- **SNS 공유**: 단체 사진 + 게임 링크 + 어린이동아 앱 공유 QR 코드 배부.

- **UGC 동의서**: Filamentree가 양식 제공, 동아이지에듀 법무 검토 후 공동 사용. 만 14세 미만 보호자 서명란 별도 구성.
- **데이터 공유**: 갱신율 데이터는 익명 집계 형태로 90일마다 상호 공유.

## 8. Success Metrics

### 구독 락인 KPI (핵심)

| 지표 | 측정 방법 | 목표값 |
|------|---------|-------|
| **참가자 구독 갱신율** | 참가 후 60일 내 갱신율 vs 대조군(미참가 구독자 동기 코호트) | 참가자 85%+ (대조군 대비 +15%p) |
| **비구독자 → 구독 전환율** | 비구독자 참가자 중 3개월 내 신규 구독 가입 비율 | 20%+ |
| **구독 해지 방어율** | 워크숍 참가자 3개월 내 해지율 vs 미참가자 | 미참가 대비 50% 감소 |

| 지표 | 측정 방법 | 목표값 |
