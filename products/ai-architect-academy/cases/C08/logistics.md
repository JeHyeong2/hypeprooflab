## 1. 공공 시설 평가 체크리스트 (Venue Assessment)

### 1.1 사전 답사 필수 항목

공공 시설은 동아미디어센터와 달리 IT 인프라 수준이 극도로 가변적이다. **모든 시설은 D-30 이전에 반드시 현장 답사를 실시한다.**

| 항목 | 최소 요건 | 권장 사양 | 체크 방법 | 불합격 시 대응 |
|------|---------|---------|----------|-------------|
| **면적** | 30평 이상 (10팀 기준) | 50평 이상 (15팀) | 실측 또는 도면 확인 | 팀 수 축소 (10팀 이하) |
| **좌석** | 팀당 2인 착석 가능 테이블 | 팀당 독립 테이블 (1.2m×0.6m) | 현장 확인 | 학교 책상 추가 반입 요청 |
| **전원 콘센트** | 10구 이상 (벽면 콘센트) | 20구 이상 | 실측 — 벽면 콘센트 위치·수량 기록 | **자체 멀티탭 10개 + 연장 케이블 5개 반입 (필수)** |
| **Wi-Fi** | 30Mbps 이상, 동시 20대 | 50Mbps 이상, 동시 30대 | 스마트폰 3대 동시 속도 측정 (speedtest.net) | **자체 LTE 라우터 3대 반입 (필수)** |
| **프로젝터** | 1대 (HDMI 지원) | 1080p 이상, 밝기 3000루멘+ | HDMI 연결 테스트 | 자체 포터블 프로젝터 반입 |
| **마이크** | 1개 (유선 OK) | 무선 2개 (강사+발표용) | 음량 테스트 | 자체 블루투스 스피커 + 유선 마이크 반입 |
| **냉난방** | 정상 작동 | 온도 조절 가능 | 에어컨/보일러 작동 확인 | 계절에 따라 선풍기/전열기 반입 |
| **화장실** | 시설 내 또는 100m 이내 | 50m 이내 | 현장 확인 | 참가자 안내도 제작 |
| **주차** | 건물 내 또는 인근 공영 | 무료 주차 | 주차 가능 대수 확인 | 대중교통 안내 강조 |
| **조도** | 프로젝터 투사 가능 수준 | 암막 커튼 있음 | 오후 3시 기준 빛 유입 확인 | 자체 암막 천 반입 (₩20,000) |

### 1.2 시설 등급 분류

답사 결과를 기반으로 시설을 3등급으로 분류하여 장비 반입 수준을 결정한다.

| 등급 | 기준 | 자체 반입 필요 항목 | 해당 시설 예시 |
|------|------|-----------------|-------------|
| **A (양호)** | Wi-Fi 50Mbps+, 콘센트 15구+, 프로젝터 있음 | 멀티탭 5개, USB 10개 | 대형 청소년 수련관, 신축 도서관 |
| **B (보통)** | Wi-Fi 30Mbps 미만 또는 콘센트 10구 미만 | LTE 라우터 2대 + 멀티탭 10개 | 구립 도서관 다목적실, 구형 수련관 |
| **C (열악)** | Wi-Fi 없음 또는 콘센트 5구 미만 | LTE 라우터 3대 + 멀티탭 10개 + 연장 케이블 5개 + 포터블 프로젝터 | 주민센터 소회의실, 노후 교육실 |

> **C등급 시설은 15팀 운영 불가 — 10팀 이하로 축소하거나 대체 시설 요청.**

## 6. 장비 비상 대응 (Equipment Contingency)

### 6.1 공공 시설에서 자주 발생하는 문제

| 문제 | 발생 빈도 | 즉시 대응 |
|------|----------|----------|
| **Wi-Fi 없음 또는 속도 5Mbps 미만** | 매우 높음 | LTE 라우터 3대 즉시 가동 → A3 출력물로 SSID/PW 안내 |
| **콘센트 부족 (5구 미만)** | 높음 | 멀티탭 10개 + 연장 케이블 5개로 벽면 콘센트에서 분배 |
| **프로젝터 미작동** | 중간 | 포터블 프로젝터 교체 (2분 소요) 또는 강사 노트북 화면 공유 (참가자 각자 화면 확인) |
| **참가자 노트북 고장** | 중간 | 대여 노트북 10대 중 교체 지급 |
| **에어컨/히터 고장** | 낮음 | 창문 개방 + 휴식 시간 외부 공기 순환 / 겨울은 참가자 방한복 사전 안내 |
