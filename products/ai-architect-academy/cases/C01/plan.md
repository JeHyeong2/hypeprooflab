# C01 — 동아미디어센터 자체 정기 워크숍 (Direct B2C)
## Full Business Plan

---

## Executive Summary

동아일보가 이미 보유한 서울 종로구 동아미디어센터 교육장, 어린이동아·동아닷컴 홍보 채널, 동아이지에듀 구독자 네트워크를 100% 활용하여 추가 인프라 투자 없이 초등 4~6학년과 학부모를 대상으로 유료 B2C AI 교육 워크숍을 월 1~2회 운영하는 파일럿 사업이다. Filamentree(HypeProof AI Lab)가 커리큘럼과 강사를 제공하고 동아이지에듀가 운영을 분담하며, 1회당 ₩200,000/팀 × 15~20팀 기준 매출 ₩3,000,000~4,000,000, 비용 ₩1,450,000~1,650,000의 50%+ 마진 구조를 갖춘다. 파일럿 1회(2026-03-28) 완료 후 NPS 90%+ 달성을 기준으로 정기화 의사결정을 진행하며, 광고 수익 -11.3% YoY 하락 국면에서 '동아일보 AI Academy' 브랜드를 앵커로 한 새로운 B2C 수익원을 즉각 확보한다. 계약 체결은 신청 오픈 전 필수이며, 미성년자 참가자 전원에 대한 법정 개인정보·촬영 동의는 운영의 선결 조건이다.

## 1. Case Context

### 1.1 장소 — 동아미디어센터 교육장 (서울 종로구)

| 항목 | 내용 |
|------|------|
| 위치 | 서울특별시 종로구 청계천로 1 동아미디어센터 내 교육장 |
| 공간 규모 | 60~80평, 20개 팀 테이블 배치 가능 |
| 운영 시간 | 주말(토/일) 13:00~17:00, 평일 공실 시간 활용 |
| 임대료 | 없음 — 동아일보 직영 시설, 내부 조율로 공실 활용 |
| 접근성 | 지하철 1·2·5호선 광화문·시청역 도보 5분 이내 |

### 1.2 참가자 — 초등 4~6학년 + 학부모 1:1 팀

| 항목 | 내용 |
|------|------|
| 연령 | 초등 4학년(만 9~10세) ~ 6학년(만 11~12세) |
| 팀 구성 | 자녀 1명 + 학부모 1명 = 1팀 |
| 정원 | 15~20팀/회 |
| 기술 수준 | 코딩 경험 없어도 무관, 스마트폰·태블릿 조작 가능 수준 |
| 보호자 역할 | Integrator — 아이디어를 구조화하고 현실적 질문을 던지는 역할 |

**초등 4~6학년 특성 고려사항:**
- 집중 지속 시간 15~20분 단위 → Part마다 활동 전환 필요
- 경쟁보다 성취감이 동기 유발에 효과적
- 학부모가 자녀보다 AI에 더 낯설 가능성 높음 → 동등한 학습자로 설정
- 성공 경험(게임 완성)이 핵심 — 미완성팀 발생 방지 위한 조교 집중 지원 필수

### 1.3 운영 제약사항

| 제약 | 내용 | 대응 |
|------|------|------|
| 동아일보 직영 운영 | 브랜드 완전 통제, 외부 스폰서 노출 불가 | Filamentree 강사진은 "동아 AI Academy 강사"로 소개 |
| 마케팅비 최소화 | 기존 채널(어린이동아, 동아닷컴)만 사용 | 어린이동아 뉴스레터 + 동아닷컴 교육섹션 배너 활용 |
| 월 1~2회 주말 운영 | 교육장 예약 시스템 내 공실 확보 필요 | 최소 4주 전 예약 확정 필수 |
| 미성년자 참가자 | 14세 미만 법정대리인 개인정보·촬영 동의 법적 필수 | 신청 Form 동의 항목 + 현장 서면 동의 이중 확보 |

## 2. Curriculum Adaptation

### 2.1 원칙: 4-Part 구조 완전 보존

기본 커리큘럼의 4-Part 구조(40분 + 50분 + 90분 + 60분 = 240분)를 변경하지 않는다. 초등 4~6학년에 맞게 **예시·어휘·속도**만 조정한다.

### 2.2 Part별 적응 상세

#### Part 1: 관점의 전환 (Mindset Shift) — 40분 (13:00~13:40)

**시간 조정:** 변경 없음 (40분)

**콘텐츠 수정:**

| 원본 (기본 커리큘럼) | C01 적응 (초등 4~6학년) |
|---------------------|----------------------|
| "노동자 vs 건축가" 프레이밍 | "마인크래프트 유저 vs 마인크래프트 개발자" 비유 — 블록 쌓는 사람 vs 세계 규칙을 설계하는 사람 |
| "설계도면을 그리는 사람" | "게임 설명서를 쓰는 사람" — 초등생에게 친숙한 표현 |
| AI 라이브 데모 (30초 게임 생성) | 동일 유지 — 단, 시연 게임을 인기 초등 게임(공룡 점프, 벽돌 깨기)과 유사한 장르로 선택 |
| "코딩 문법" 언급 | 완전 제거 — "AI한테 말로 시킨다"로 대체 |
| 팀 이름 정하기 | "게임 회사 이름 정하기"로 변경 → 더 강한 몰입감 유발 |

**어휘 가이드:**
- "Context Engineering" → "AI한테 잘 설명하기"
- "PRD (기획서)" → "게임 설계도"
- "Architect" → "게임 세계 설계자"
- "Visionary" → "아이디어맨/아이디어걸"
- "Integrator" → "구체화 담당"

#### Part 2: 협업과 조율 (Collaboration) — 50분 (13:40~14:30)

**시간 조정:** 변경 없음 (50분)

**콘텐츠 수정:**

| 항목 | 원본 | C01 적응 |
|------|------|----------|
| 샘플 게임 분석 | 일반 데모 게임 | 이전 참가자가 만든 실제 게임 사용 (ryoo8099.github.io/saving-my-own-cat 등) — "이거 초등학생이 4시간 만에 만든 거예요" 임팩트 |
| 브레인스토밍 프롬프트 | 자유 발산 | "내가 제일 좋아하는 게임은?" "거기서 가장 재미있는 부분은?" → 아는 것에서 출발 |
| 기획서(PRD) 양식 | 표준 양식 | 그림 그릴 칸 추가 (글 쓰기 어려운 아이를 위한 시각적 기획서 옵션) |
| 부모 역할 안내 | 구두 설명 | 별도 "학부모 미션 카드" 배부 — "지금 이 순간 학부모님은 아이의 첫 번째 동료입니다" |

**초등생 집중력 관리 포인트:**
- 브레인스토밍 15분을 8분(자유 발산) + 7분(부모 정리)로 명확히 분리
- 기획서 작성 10분: 필수 항목 3개로 축소 (게임 이름, 주인공, 이기는 방법) — 완성 부담 최소화

#### Part 3: 지휘와 소통 (Directing) — 90분 (14:30~16:00)

**시간 조정:** 변경 없음 (90분)

**콘텐츠 수정:**

| 항목 | 원본 | C01 적응 |
|------|------|----------|
| 맥락 설계 훈련 | "그냥 해줘" vs 구체적 지시 비교 | 카드게임 방식 도입 — "나쁜 지시 카드" vs "좋은 지시 카드"를 뒤집어가며 체험 (게임화) |
| 실습 1차 기본 구현 | 자유 진행 | 3단계 체크리스트 제공: 1) 캐릭터 만들기 2) 이동 시키기 3) 점수 추가하기 — 막히면 체크리스트 따라가기 |
| 10초 피드백 루프 | 자율 진행 | 조교가 팀당 순회 주기를 10분으로 고정 — 초등생은 막혀도 손 들기 어려워 능동적 순회 필수 |
| "포기의 미덕" 훈련 | 추상적 개념 | "오늘의 MVP(Most Valuable Part)" 스티커 — 완성할 기능에 스티커, 포기할 기능은 "다음 버전으로" 스티커 |
| 실습 2차 개선 | 자유 개선 | 15:30에 전체 대상 "5분 남았을 때 뭘 할지" 발표 → 완성 목표 명확화 |

**추가 조교 지원 기준:**
- 초등 4학년(만 9~10세): 조교 1명이 3팀 이하 담당 (가장 어린 학년)
- 초등 5~6학년: 조교 1명이 4~5팀 담당
- 글자 입력 속도가 느린 아이 → 학부모가 타이핑, 아이가 구두로 지시하는 방식 허용

#### Part 4: 성취와 비전 (Achievement) — 60분 (16:00~17:00)

**시간 조정:** 변경 없음 (60분)

**콘텐츠 수정:**

| 항목 | 원본 | C01 적응 |
|------|------|----------|
| 팀별 발표 (3~5분) | 동일 유지 | 발표 도우미 카드 제공 — "이 게임은 ___하는 게임입니다. 제가 가장 힘들었던 건 ___였어요." |
| 시상 5개 카테고리 | 동일 유지 | 상품을 문구류+도서상품권으로 통일 — 학부모 취향보다 아이 취향에 맞춤 |

| 클로징 메시지 | 일반 비전 | "여러분이 만든 게임은 지금 인터넷에 올라가 있어요. 전 세계 누구나 플레이할 수 있어요" — 구체적 성취 강조 |
| 피드백 설문 | 구두 설문 | QR코드 기반 온라인 설문 (Google Form) — 학부모용 NPS 7문항, 아이용 이모지 만족도 3문항 |

**사진/영상 콘텐츠 계획 (동아일보 채널 활용):**
- 게임 시연 순간 영상 촬영 → 어린이동아 SNS 릴스 콘텐츠
- 수료증 수여 단체 사진 → 동아닷컴 교육 기사 삽입
- 학부모 소감 인터뷰 (개인정보·촬영 동의서 서명 완료자만)
