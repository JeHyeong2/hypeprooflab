# AI Architect Academy — Product Specification

> **Turn kids into AI Commanders, not Code Monkeys.**

---

## 1. Product Vision

### 1.1 One-Liner
**"부모-자녀가 함께 AI를 지휘하며 진짜 게임을 만드는 체험형 학습 플랫폼"**

### 1.2 Philosophy (HypeProof Core)
```
AI solves problems. Humans define them.
→ 코딩은 AI가 합니다. 당신의 아이는 '무엇을 할지' 결정만 하면 됩니다.
```

### 1.3 Market Gap

| 기존 시장 | 우리의 포지션 |
|-----------|--------------|
| 코딩 학원: 문법 암기 위주 | **설계적 사고** 훈련 |
| AI 교육: 이론 중심, 결과물 없음 | **4시간 안에 게임 완성** |
| 개인 학습: 고립된 경험 | **부모-자녀 협업** (Co-Founding) |
| 값비싼 장기 과정 | **1회성 집중 워크숍** (₩150,000~300,000) |

## 2. Product Strategy

### 2.2 Phase 1: Offline Workshop Business

**Target**: 동아일보 파트너십으로 검증된 오프라인 워크숍

| 항목 | 내용 |
|------|------|
| **형태** | 4시간 부모-자녀 워크숍 |
| **가격** | ₩200,000/팀 (부모1+자녀1) |
| **정원** | 15~20팀/회 |
| **수익** | ₩3,000,000~4,000,000/회 |
| **원가** | 강사비 + 장소 + 간식 ≈ ₩1,500,000 |
| **마진** | 50%+ |
| **빈도** | 월 2회 (서울) → 월 4회 (서울+지방) |

**Phase 1 목표**:
- [ ] 동아일보 1차 워크숍 성공적 진행
- [ ] 참가자 NPS 70+ 달성
- [ ] 최소 3개 미디어 파트너 확보 (대기업 사보, 지역 신문 등)
- [ ] 월 매출 ₩8,000,000 (4회 × 20팀) 달성

### 2.4 Phase 3: Full Online SaaS

**"언제 어디서나 부모-자녀가 AI 프로젝트를 만드는 플랫폼"**

#### 핵심 차별점

1. **실시간 협업 캔버스** — 부모와 자녀가 같은 화면에서 아이디어 정리
2. **AI 코파일럿** — 연령/레벨에 맞춰 AI 응답 조절 (초급: 친절, 고급: 간결)
3. **프로젝트 템플릿** — 게임, 앱, 웹사이트, 자동화 등 다양한 시작점
4. **성장 트래킹** — 완성 프로젝트 포트폴리오, 스킬 트리, 배지 시스템

#### Pricing Tiers

| Tier | 가격 | 포함 내용 |
|------|------|----------|
| **Free** | ₩0 | 월 1개 프로젝트, 기본 템플릿, 워터마크 |
| **Family** | ₩29,000/월 | 무제한 프로젝트, 모든 템플릿, AI 크레딧 100K토큰/월 |
| **Pro** | ₩79,000/월 | Family + 팀 협업, 고급 AI 모델, 우선 지원 |
| **School** | 별도 문의 | 학교/학원용 관리 대시보드, 학급 라이선스 |

## 4. Tech Stack (Proposed)

### 4.1 Frontend
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + Framer Motion
- **Real-time**: Liveblocks (협업 캔버스) 또는 Socket.io

### 4.2 Backend
- **Runtime**: Node.js (Bun 고려)
- **API**: tRPC or REST
- **DB**: Supabase (PostgreSQL + Auth + Storage)
- **AI**: OpenAI API + Claude API (멀티 모델 지원)

### 4.3 Infrastructure
- **Hosting**: Vercel (Frontend) + Railway/Fly.io (Backend)
- **CDN**: Cloudflare
- **Analytics**: PostHog (프라이버시 친화)
- **Payment**: Toss Payments (국내) + Stripe (해외)

### 4.4 AI Coding Environment
- **Primary**: Antigravity IDE (검증 필요)
- **Fallback**: Custom Browser IDE (Monaco + WebContainer)
- **Long-term**: 자체 AI Coding Playground 개발

## 9. Financial Projections (Year 1)

### Revenue Streams

| 항목 | Q1 | Q2 | Q3 | Q4 | 연간 |
|------|-----|-----|-----|-----|------|
| 워크숍 (B2C) | ₩6M | ₩12M | ₩16M | ₩16M | ₩50M |
| 기업 워크숍 (B2B) | - | ₩5M | ₩10M | ₩15M | ₩30M |
| 플랫폼 구독 | - | - | ₩2M | ₩6M | ₩8M |
| B2B 라이선스 | - | - | - | ₩5M | ₩5M |
| **합계** | **₩6M** | **₩17M** | **₩28M** | **₩42M** | **₩93M** |

### Cost Structure

| 항목 | 월 고정비 | 변동비 (워크숍당) |
|------|----------|-----------------|
| 인건비 (파트타임) | ₩3,000,000 | - |
| 강사비 | - | ₩500,000 |
| 장소/간식 | - | ₩300,000 |
| 마케팅 | ₩1,000,000 | ₩200,000 |
| 인프라 | ₩500,000 | - |
| **합계** | **₩4,500,000** | **₩1,000,000** |

### Break-even
- 워크숍 가격 ₩200,000/팀 × 15팀 = ₩3,000,000/회
- 변동비 ₩1,000,000/회 → 공헌이익 ₩2,000,000/회
- 월 고정비 ₩4,500,000 ÷ ₩2,000,000 = **월 3회 손익분기**
