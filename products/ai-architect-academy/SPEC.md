## 3. 문제 정의 (Problem Finding)

### Gap 1: 코딩 기술자 ≠ AI 시대 인재
- 2028년 개발자 75%가 AI 코딩 도구 사용 (Gartner)
- 기존 코딩 취업률 90% → 60% 급락 전망
- "프롬프트 엔지니어링이 새로운 코딩이다"

### Gap 2: 아이 혼자 하는 교육의 한계
- 기존 학원: 아이만 수강, 부모는 대기실 방치
- 부모가 모르면 가정 내 확장 불가
- '함께 배우는' 모델의 부재

### Gap 3: 만드는 경험 부재
- 학교 정보교육 연간 34시간 불과
- 이론 위주 교육, '내 게임' 완성 성취감 부족

> **핵심 질문**: 코딩 문법을 외우게 할 것인가, AI를 지휘하는 법을 알려줄 것인가?

## 4. 핵심 컨셉 (Core Concept)

### 4.1 비전
**코딩 기술자가 아닌, AI 시대를 지휘할 설계자(Architect)를 키웁니다.**

### 4.2 부모-자녀 Co-Founding 모델

```
┌─────────────────────────────────────────────────┐
│              하나의 팀 = 하나의 스타트업              │
│                                                 │
│   👧 자녀 (Visionary)     👨 부모 (Integrator)    │
│   ┌─────────────────┐    ┌──────────────────┐   │
│   │ 무한한 상상력     │    │ 현실 감각         │   │
│   │ "하늘 나는 고양이" │    │ "날개는 어떤 버튼?" │   │
│   │ 아이디어 발산     │    │ 구조화·구체화      │   │
│   │ AI 직접 지휘     │    │ 기획서 정리 지원    │   │
│   └─────────────────┘    └──────────────────┘   │
│                    ↕ 충돌·융합                     │
│              🤖 AI (Executor)                    │
│         지시받은 대로 코드를 작성하는 실행자           │
└─────────────────────────────────────────────────┘
```

### 4.3 핵심 역량 3가지
1. **맥락 설계 (Context Engineering)** — AI에게 올바른 지시를 내리기 위한 논리적 사고
2. **의사결정 (Decision Making)** — 무엇을 만들고, 무엇을 포기할지 판단하는 능력
3. **협업 리더십 (Collaborative Leadership)** — 사람(부모)과 AI를 동시에 지휘하는 경험

## 5. 커리큘럼: 4시간, 4단계 (Action Plan)

### Part 1: 관점의 전환 (40min) 🔄
- AI 라이브 데모
- 노동자 vs 건축가 토론
- "내가 직접 벽돌을 나를 것인가, 설계도면을 그릴 것인가?"

### Part 2: 협업과 조율 (50min) 🤝
- 샘플 게임 분석
- 기획서 작성 (아이: 상상 / 부모: 정리)
- 아이: "하늘 나는 고양이를 만들래" → 부모: "날개는 어떤 버튼으로?"

### Part 3: 지휘와 소통 (90min) 🎮
- AI 지휘하여 실제 게임 제작
- **10초 피드백 루프** (즉각적 결과 확인)
- 맥락 설계(Context Engineering) 실전 훈련

### Part 4: 성취와 비전 (60min) 🏆
- 팀 발표
- 시상 (5개 카테고리 — 모든 팀 수상)
- 수료증 수여

### 특징
| 요소 | 설명 |
|------|------|
| 🔄 10초 피드백 루프 | AI에게 지시 → 즉시 결과 확인 → 수정 → 반복 |
| 🎯 포기의 미덕 | 핵심에 집중하는 훈련 (스코프 조절 능력) |
| 🏆 5개 시상 카테고리 | 모든 팀이 하나는 수상 (성취감 보장) |

## 6. 동아일보 파트너십 구조

### 6.1 동아일보의 AI 교육 포트폴리오 빈 칸

| 기존 프로그램 | 대상 | 수익 모델 |
|-------------|------|----------|
| 미디어 프론티어 부트캠프 | 취준생 (성인) | 국비지원 (K-Digital) |
| 동아이지에듀 (어린이동아, 신나는공부) | 초등~고등 | 구독료 + 광고 |
| DBR에듀 | 직장인/경영진 | 수강료 |

**빈 칸: 초중등 학생 + 학부모 대상 유료 AI 교육 프로그램이 없습니다.**

조선일보는 에듀조선(뉴지엄, 기자체험관)으로 K-12 체험 교육을 이미 운영 중. 2025년 코딩교육 필수화, 2026년 AI 디지털교과서 도입으로 학부모의 AI 교육 수요가 폭발하는 지금, 이 빈 칸을 먼저 채우는 언론사가 시장을 선점합니다.

### 6.2 동아일보가 하는 것 vs 얻는 것

#### 동아일보가 하는 것 (이미 가진 자산 활용)

| 기여 항목 | 내용 | 추가 비용 |
|----------|------|----------|
| 브랜드 | "동아일보 × Future AI Leader's Academy" 공동 명칭 사용 | 없음 |
| 공간 | 동아미디어센터 또는 기존 교육장 활용 (월 1~2회) | 없음 |
| 홍보 | 어린이동아·동아닷컴·교육섹션 배너 및 기사 | 기존 채널 |
| 네트워크 | 동아이지에듀 구독자·미디어프론티어 수료생 연계 | 없음 |

**핵심: 동아일보는 추가 인력·예산 투입 없이, 이미 보유한 자산만으로 참여합니다.**

#### 동아일보가 얻는 것

| 성과 | 설명 |
|------|------|
| **AI 교육 브랜드** | "동아일보 AI Academy" → 조선일보 에듀조선과 차별화 |
| **B2C 신규 매출** | 워크숍 수익의 일부 (수수료 또는 정액) |
| **콘텐츠** | 워크숍 기반 기사, 영상 콘텐츠 무상 확보 |
| **고객 데이터** | 학부모/학생 관심사 데이터 → 구독 연계 |
