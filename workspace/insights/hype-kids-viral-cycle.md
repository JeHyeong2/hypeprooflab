# Hype Kids — Viral Cycle Strategy

> 작성: 2026-04-22
> 맥락: `/kids-edu` 프로그램을 이수한 아이들이 자연스럽게 HypeProof 커뮤니티에 스며들고, 그들을 통해 주변 아이들이 흡수되는 순환 사이클 설계
> 1차 타겟: 부유층 자녀 (초등 고학년 ~ 중학생, 10-15세)

---

## 1. Target Reality Check

부유층 자녀 시장을 정확히 이해해야 설계가 무너지지 않는다.

### 이 아이들의 재화 희소성
- 물질: **이미 다 있음** — 최신 기기, 해외 여행, 국제학교 / 특목, 5-7개 extracurricular. 장난감/게임 보상은 먹히지 않음.
- 시간: **공급 측이 조임** — 수학·영어·악기·스포츠로 주 7일 채워짐. HypeProof는 시간 경쟁에서 이겨야 함.
- **"내가 만든 것"** 은 이 아이들 일상에서 희귀 자원이다. 모든 게 **사 준 것, 시켜서 한 것** 이기 때문. → **직접 만든 결과물 = 강력한 정체성 통화**.

### 부모의 의사결정 구조
- 부모가 입구를 연다, 아이가 체류를 결정한다.
- 부모 선택 기준:
  1. **포트폴리오 가치** (영재고/국제학교/대학 prep)
  2. **또래 엄마 네트워크에서 이야기할 만한 프로그램** (맘카페 강남·분당·서초)
  3. **너무 mass market은 아닐 것** (exclusivity signal)
  4. **미래 지향성** (AI, creative, leadership — 10년 후 통할 것)
- HypeProof는 이 4박자를 이미 가지고 있음. 문제는 **인지도**.

### 학교 월요일 아침이 진짜 전장이다
- 초·중생이 가장 많은 시간을 보내는 **또래 집단**은 학교 반/학년.
- 월요일 아침 "주말에 뭐 했어?" 질문 = 정체성 증명 기회.
- 여기서 **자랑할 것이 있는 아이** = 반에서 특별해짐 → 친구가 관심 → 친구 부모에게 소식 전달 → 다음 기수 유입.

**따라서**: 전체 viral cycle의 핵심은 월요일 아침 **교실에서 일어나는 10분**에서 결판난다.

---

## 2. The "Monday Morning Moment" — 중앙 허브

아이들 대상 viral cycle에서 가장 과소평가되는 모먼트.

### 이 10분이 작동하려면 필요한 3가지

1. **즉석에서 보여줄 수 있는 증거** — 친구 핸드폰에 QR로 5초 만에 내 게임 로드
2. **내가 주인공인 공유 자료** — 대형 스크린에서 상영된 내 얼굴 + 게임 영상 (학교에서 틀 수 있는 포맷)
3. **"어떻게 만들었어?" 질문에 답할 프레임** — 아이가 평소 쓰는 언어로 된 간단한 자기 소개 스크립트 ("나 지휘관 됐어", "AI한테 시켜서 만든 거야")

### 우리가 준비해 세션 당일 쥐어줘야 할 것들

- [x] **개인 QR 마스터 키** — 어떤 스마트폰에서도 바로 내 게임 로드 (이미 curriculum v2.1에 있음)
- [x] **"마스터 개발자 OOO" 사원증** (이미 있음) — 목걸이 형태 / NFC 내장도 검토
- [ ] **게임 런칭쇼 영상 클립 30초** — 세션 당일 촬영 → 금요일까지 편집 → 월요일 등교 전 아이 카톡으로 발송
- [ ] **"친구에게 내 게임 소개하는 법" 1페이지 카드** — 아이 언어로 3문장, 노는 친구들 사이에서 어색하지 않게
- [ ] **"너도 해볼래?" 친구 초대 QR** — 사원증 뒷면. 친구 부모 1인 무료 상담 트리거

## 3. 7가지 Hooking Moments (phase별)

viral cycle 단계별로, 각 단계에서 아이가 자발적으로 다음 단계로 넘어가게 만드는 장치.

### Phase A — Entry (부모가 문을 엶)

**Hook 1. "한정 기수" 장치**
- 기수당 **40명 한정** 프레임 유지 (어린이날 NCC 세션 규모 유지).
- 신청서 도입: "왜 참여하고 싶은지 부모/자녀 3문장씩". 선발 과정을 둠으로써 **exclusivity signal**.
- 부유층 부모가 맘카페에 "우리 OO 이거 됐어요" 라고 자랑할 만한 프레임 구축.

**Hook 2. 부모 네트워크 레퍼럴 구조**
- 부모 2인이 같이 신청 시 소정 할인 OR 우선권.
- 세션 후 부모용 1페이지 리포트: "우리 아이가 어떤 설계 사고를 보였는지" — 다른 엄마들 대화에서 자연스럽게 회람되는 자료.

### Phase B — Hook (세션 당일)

**Hook 3. 대형 스크린 개인 상영**
- 40명 중 내 게임이 대형 스크린에 이름과 함께 상영되는 순간 = 평생 기억되는 시각 체험.
- 부모가 이 장면을 촬영해 저장 → 가족 단위 자료로 누적.

**Hook 4. 사원증 & 실물 패키지**
- "마스터 개발자" 사원증, 실물 게임 패키지 = **만질 수 있는 증거**.
- 저가형이 아닌 **premium** — 레더 스트랩, 홀로그램 스티커, 개별 시리얼 넘버. 부유층 아이 눈높이에 맞춰.

### Phase C — Monday Morning (교실)

**Hook 5. "친구야, 내가 만든 게임 해볼래?"**
- 앞에서 설명한 중앙 허브 모먼트.
- 즉석 플레이 → "와 이거 진짜 돌아가네" 반응 → 학급 내 화제.

**Hook 6. 자랑용 소셜 자료 패키지**
- 아이용 : 게임 런칭 30초 영상 (자기 얼굴 + 게임 + 박수소리) — 카톡 프로필 뮤직 or SNS 스토리 업로드
- 부모용 : 세션 비하인드 + 우리 아이 인터뷰 20초 + 결과물 — 인스타 스토리 포맷
- **양측 다 준비됐을 때 가장 빠르게 확산**

### Phase D — Absorption (커뮤니티 합류)

**Hook 7. "Hype Kids" 전용 공간**
- 기수 수료자만 입장 가능한 Discord/플랫폼 서브 공간.
- 주간 가벼운 미션 ("이번 주는 엔딩 있는 게임 만들기"). 집에서 30분이면 됨.
- 기수 간 배틀 (게임끼리 investing, peer review).
- **2기 수료자 = 3기 멘토**. 아이가 남의 작품에 코멘트하는 경험 → 공동체 주도성.

### Phase E — Recruitment (다음 기수 유입)

**Hook 8. "나랑 같이 가자" 친구 초대 코드**
- 사원증 뒷면 QR = 친구 1인 초대 권한.
- 초대받은 아이가 신청 시 **둘이 한 팀** (Doubles 구조)으로 다음 세션 참여.
- 친구가 진짜 오면 초대한 아이에게 **"Senior 지휘관" 업그레이드 배지** (경쟁 아닌 인정).

## 4. 설계 원칙 (affluent context-specific)

1. **premium aesthetic** — 세션 공간, 영상, 패키지가 싸구려 느낌 1% 도 나지 않도록. Apple 라이트 톤이 이미 올바른 방향.
2. **exclusivity signal**: 숫자로 보여라 (40명, 기수당 1회). 희소성은 affluent 부모가 지갑을 여는 가장 강한 신호.
3. **"돈 없어도 올 수 있는 프로그램" 아님을 명시하지 말되, 그것을 느끼게는 할 것** — 직접적으로 비싸다고 말하면 tacky. 간접적으로 quality·exclusivity·curation으로 전달.
4. **학교를 스킵하지 말고 학교를 활용하라** — 교내 동아리 연계 / 반 단위 파일럿 / 학교 행사 런칭. 학교가 확산 노드.
5. **부모-자녀 동시 노출** — 부모용 콘텐츠와 자녀용 콘텐츠를 분리해 동시 제작. 부모는 성인 네트워크에, 자녀는 또래 네트워크에 각자 전파.
6. **디지털 + 물리 이중 증거** — QR(디지털)과 사원증(물리)이 서로를 보완. 한 쪽만 있으면 약함.
7. **경쟁을 감추고 성장을 드러내라** — 부유층 아이들은 이미 학원에서 경쟁에 지쳐있음. "남을 이기는 경험" 대신 **"내가 작년보다 컸다"** 프레임. 시즌별 PR 기록이 자연스러움.

## 5. 안티 패턴 (하지 말 것)

- ❌ **대량 저가 마케팅** — 맘카페 배너 광고, 틱톡 광고 등. affluent 부모는 이런 채널로 들어온 프로그램을 덜 신뢰.
- ❌ **메달 남발 / 참가상** — "참가하면 누구나 받는 것" = 가치 하락. Hyrox도 finisher 메달은 강력하지만 남발하지 않음.
- ❌ **기술 중심 마케팅 copy** — "코딩을 배웁니다" 는 이미 학원에서 넘침. "AI를 지휘합니다" 가 맞는 프레임 (현재 copy 이미 OK).
- ❌ **공짜 후기 유도** — 인스타 공유하면 할인 쿠폰 식. 부유층 부모에게 유치함 신호.
- ❌ **부모 없는 단독 참여** — Co-Founding 모델을 절대 포기하지 말 것. 부모 참여가 premium 경험의 핵심 차별 + 부모 확산 네트워크의 트리거.

## 6. 실험 설계 — 어린이날 세션을 pilot으로 검증

`/kids-edu/childrens-day-2026` 세션에서 이 viral cycle의 **1차 검증 가능**:

### 세션 당일 수집
- [ ] 부모가 직접 촬영한 "지휘관 얼굴" 사진 (표준 포맷 가이드 제공)
- [ ] 게임 런칭 30초 영상 (통일 포맷 편집)
- [ ] 사원증 배포 + QR 마스터 키 작동 확인
- [ ] 부모 1페이지 리포트 초안 테스트

### 세션 후 1주일 내 측정
- [ ] 아이들이 월요일에 친구에게 보여주는 비율 (사후 인터뷰)
- [ ] 부모가 SNS / 지인에게 공유한 건수
- [ ] 친구 초대 QR 스캔 건수
- [ ] 2기 신청 의향 문의 건수

### 3개월 후 확인
- [ ] 첫 기수 중 몇 명이 자발적으로 Discord 주간 미션에 참여?
- [ ] 기수 멤버가 데려온 친구 비율?
- [ ] 학교 동아리 연계 문의 수?

**지표가 약하면**: Monday Morning 장비(영상/QR/스크립트)를 먼저 보강. 강하면: Phase D (Discord 커뮤니티 공간) 본격 오픈.

## 7. 한 줄 요약

> **"세션은 씨앗, 월요일 아침이 뿌리다."**

아이들 viral cycle은 세션 당일 멋지게 끝나서 생기는 게 아니라, **월요일 아침 교실에서 친구에게 내 게임을 보여주는 10분** 에서 만들어진다. 우리는 그 10분을 돕기 위한 **모든 물질적·디지털 증거**를 준비해야 한다.

그리고 그 10분이 성공하면, 그 아이의 친구가 다음 기수 신청서를 쓴다. 그 친구의 엄마가 맘카페에 글을 쓴다. 그 반 아이 절반이 부모에게 "나도 하고 싶어" 라고 말한다.

**교실 한 칸 = 하나의 Hype 노드.** Hyrox의 "짐"이 HypeProof에서는 **학교 반/학년**이다.
