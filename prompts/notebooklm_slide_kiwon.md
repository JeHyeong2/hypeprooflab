0) 기본 규칙

모든 텍스트는 한국어로만 작성할 것(영문, 로마자, 혼용 금지)

슬라이드에서 사용하는 문장 종결은 ‘~함 / ~임’ 중심의 명사형으로 통일할 것

불필요한 마크다운 기호/이모지 사용 최소화

1) 스타일 전략: 모던 미니멀 + 캐치테이블 톤

PDF → PPT 변환 최적화를 위해 모던 미니멀리스트 기반으로 구성하되, 캐치테이블(Catchtable) UI/브랜드 무드처럼 “정돈된 정보 구조 + 여백 + 명확한 위계”를 유지할 것.

여백을 넉넉히 사용하고, 텍스트/도형/배경은 명확히 분리할 것

헤더-본문-보조정보(캡션/주석)의 타이포 위계를 확실히 만들 것

디자인 장식 요소는 최소화하고, “깔끔한 제품/서비스 소개 슬라이드”처럼 구성할 것

2) 컬러 전략: Stella Artois 메인 컬러 + Catchtable 보조 컬러

컬러는 Stella Artois 메인 컬러를 핵심 포인트 컬러로 사용하고, **Catchtable 스타일의 중립 톤(검정/회색)**을 베이스로 사용할 것.

배경: Pure White #FFFFFF 고정

메인 포인트(강조/버튼/핵심 라인): Stella Artois 레드 계열(예: #C8102E 또는 유사한 “딥 레드”)

보조 포인트(서브 강조/아이콘): Stella Artois 골드 계열(예: #D4AF37 또는 유사한 “웜 골드”)

기본 텍스트: 거의 블랙(예: #111111)

보조 텍스트/라인: 중간 회색(예: #6B7280, #E5E7EB 등)
원칙:

한 슬라이드에서 컬러는 화이트 + 블랙/그레이 + 레드(필수) + 골드(선택) 정도로 제한할 것

레드는 “정답 표시”처럼 과하게 쓰지 말고, 핵심 포인트/CTA/키워드에만 제한적으로 사용할 것

3) 배경 전략: 순수 흰색(Pure White)

배경은 항상 #FFFFFF로 고정

그라데이션/패턴/사진 배경 금지

오브젝트 분리 및 편집 가능성을 최우선으로 할 것

4) 글꼴(Font) 전략: 가독성 중심 + 한글 최적화

OCR 및 편집을 고려하여 Noto Sans KR(또는 동급의 고딕 산세리프) 사용.

제목: Bold / 본문: Regular

자간/행간은 너무 타이트하지 않게(가독성 우선)

손글씨/장식체 금지

5) 텍스트 밀도 & 어조

한 슬라이드 텍스트 5줄 이하(권장 3줄)

문장 길이는 짧게, 키워드 중심으로

어조는 “분석적/정돈된 서비스 설명” 느낌으로, 과장/감성표현 최소화

표/리스트가 필요하면 간결한 불릿으로 구성하고, 문장형 장문 금지

6) 레이아웃 & 디자인 제약(변환 친화)

제약이 만드는 완벽함 원칙 적용:

이미지 최소화(필요 시 아이콘 수준의 단순 그래픽)

텍스트와 도형이 겹치지 않도록 배치

박스/라인/구분선은 회색 톤으로 얇게, 핵심만 레드로 강조

요소들은 “PPT에서 바로 선택/수정 가능”한 벡터 형태로 구성할 것(복잡한 합성 금지)

7) 결과물 지시(출력 형태)

슬라이드 스타일은 전체적으로 일관되게 유지

각 슬라이드는 **제목(상단) / 핵심 요약(중단) / 보조 설명 또는 데이터(하단)**의 구조를 기본으로 할 것
